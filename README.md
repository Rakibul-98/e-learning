# E-learning Website

This is an UX design of an online learning website. Design implemented by Tailwind flowBite, react count up, accordion, multi carousel and react icons.

## Core Features

- React count Up
- Accordion
- Multi carousel
- Responsive Design

## Used Technologies

- React
- Tailwind
- Google map
- React Icons
- Netlify
